# Neovim Project Configuration

This project includes a local Neovim configuration (`.nvim.lua`) that adds custom features for working with NegMAS scenarios.

## Features

### Scenario Picker (`<leader>sS`)

A Telescope picker that lets you search and browse all NegMAS scenarios in `~/negmas/app/scenarios/`.

**Usage:**
- Press `<leader>sS` in normal mode, or
- Run `:Scenarios` command

**Key Bindings:**
- `<CR>` (Enter) - Open scenario directory
- `<C-i>` - Open `_info.yaml` file for the selected scenario
- `<C-s>` - Open `_stats.yaml` file for the selected scenario

**Preview:**
The picker shows a live preview with:
- Scenario path
- Contents of `_info.yaml` and `_stats.yaml` if available
- List of YAML files in the scenario directory

## Requirements

- Neovim with Telescope plugin installed
- The `.nvim.lua` file is automatically loaded when you open any file in this project

## Setup

If using `exrc` in your Neovim config:
```lua
vim.o.exrc = true
vim.o.secure = true  -- Recommended for security
```

Or for lazy-loading with plugins like `klen/nvim-config-local`:
```lua
require('config-local').setup()
```
